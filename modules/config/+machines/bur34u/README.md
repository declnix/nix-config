## Proxy

Proxy settings are read from these host-local files:

`/etc/environment.d/90-proxy.conf`
`/etc/gitconfig.d/proxy.conf`

Install them from the current shell environment:

```sh
make
```

Expected variables:

```sh
http_proxy=...
https_proxy=...
all_proxy=...
no_proxy=...
HTTP_PROXY=...
HTTPS_PROXY=...
ALL_PROXY=...
NO_PROXY=...
```
