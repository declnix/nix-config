## Proxy

Proxy settings are read from this host-local file:

`/etc/environment.d/90-proxy.conf`

Install it from the current shell environment:

```sh
make
```

Expected variables:

```sh
http_proxy=...
https_proxy=...
all_proxy=...
no_proxy=...
HTTP_PROXY=...
HTTPS_PROXY=...
ALL_PROXY=...
NO_PROXY=...
```
