# Plan Management

Procedures for creating and maintaining project plans.

## Guidelines
1. **Initiation**: Always create a new plan in `.agents/@plans/` before starting significant implementation tasks.
2. **Naming**: Use the format `<aspect>.md` (e.g., `nvf.md`).
3. **Structure**: 
   - Start with `# Plan: <Description>`.
   - Include `## Goal`, `## Detailed changes`, and `## Verification`.
4. **Lifecycle**: A plan is a living document; update it as the implementation evolves or requirements change.
